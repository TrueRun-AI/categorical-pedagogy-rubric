# Categorical Pedagogy Rubric

### A 10-Parameter Category-Theoretic Rubric for Objective Grading of LLM Explanations

This repository presents a computational rubric inspired by category theory to objectively measure and enhance the structural quality of STEM explanations in large language models.

**Core Diagnostic Example**: "Why is the sky blue?"  
- Baseline LLM response: ~70% structural coherence  
- Rubric-enhanced: 96–98% across all parameters
