# categorical-pedagogy-rubric
### A Category-Theoretic Rubric for Pedagogical Excellence  

This repository presents a 10-parameter computational rubric, inspired by category theory, for objectively measuring and enhancing the structural quality of LLM explanations in STEM domains.

**Core Diagnostic**: "Why is the sky blue?"  
- Baseline (typical LLM response): ~70% coherence  
- Rubric-enhanced: 96–98% across all parameters
